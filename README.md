# CatNap Platformer

`CatNap Platformer` — это 2D-платформер на `TypeScript + Phaser + Vite`. В ходе работы была использованна такая нейросеть, как Open AI Codex. Работа выполнена за 16 часов.

Текущая версия включает:
- видеозаставку перед началом игры;
- один игровой уровень с платформами, монетами, врагами, финишной зоной, состояниями победы и поражения;
- систему из трех жизней с отображением сердечек в интерфейсе;
- HUD, стартовое окно, паузу и финальные экраны;
- перезапуск уровня и возврат на стартовый экран.


## Запуск

Откройте папку проекта:

```bash
cd Platformer-Game-CatNap_JavaScript
```

Установите зависимости:

```bash
npm install
```

Запустите проект в режиме разработки:

```bash
npm run dev
```

Соберите production-версию:

```bash
npm run build
```

Запустите локальный просмотр production-сборки:

```bash
npm run preview
```

## Полезные команды

```bash
npm run typecheck
npm run lint
npm run format
npm run format:write
```

## Управление

- `A / D` или `Left / Right` — движение
- `W / Up / Space` — прыжок
- `Esc / P` — пауза
- `Enter / Space` — подтверждение / старт / перезапуск в overlay-окнах
- `R` — перезапуск уровня
- `B` — возврат на стартовый экран

## Структура проекта

- `src/` — исходный код игры
- `public/assets/` — изображения и видеоассеты
